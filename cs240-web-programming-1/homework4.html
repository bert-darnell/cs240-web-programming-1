Simple CUDA Sample

There is open-source cuda-samples repository maintained by NVIDIA on github. Their toolkit has one of the simplest ways to validate the CUDA installation. If you need to install CUDA click here, otherwise here is instructions for a sample application.
  1. First make a directory for building the project
    • First we will create and move into the directory to install (make) CUDA on 
      mkdir ~/Dev
    • Navigate into the directory 
      cd ~/Dev
  2. Next we will clone the repository
    • Download samples from GitHub 
      git clone https://github.com/nvidia/cuda-samples

      NOTE: Propper depedencies for make and git might be needed

  3. Next we will build the application by 'make'
    • First we will move into one of the Samples directories 
      cd ~/Dev/cuda-samples/Samples/1_Utilities/deviceQuery
    • Next we will build with make 
       class="code">make
    • Once complete, run the application 
      ./deviceQuery
  4. Enjoy enabled GPU acceleration on Ubuntu on WSL2 with the NVIDIA CUDA Platform
